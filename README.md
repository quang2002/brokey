## II. Components
* 64 switches
* 64 Diodes
* 1 Oled 128x32 - I2C
* 1 Arduino ProMicro Processor

* Case:
  * Top Layer: 320x116x5 mm
  * Plate Layer: 320x116x2 mm
  * Bottom Layer: 320x116x5 mm
  * Using 8 screws to connect layers
